## create a vertual enviorment

    python -m venv env

## activate vertual enviorment

### windows

    env\Scripts\activate

### Ubuntu

    source env/bin/activate

## install pillow

    pip install pillow

## install tkinter:-
### windows
    pip install tk
### Ubuntu
    sudo apt-get install python3-tk 
### MacOS
    brew install python-tk

## Final step

    python frontpage.py
